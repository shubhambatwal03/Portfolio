# Shubham Batwal — Portfolio

A modern, responsive personal portfolio built with React (Vite) and Tailwind CSS.  
Features a dark / light theme, project showcase, timeline-style overview, skills with external docs links, contact section and a downloadable resume. Designed to be clean, accessible and mobile-first with an amber accent color throughout.

## Features

- Responsive layout (mobile → desktop)
- Dark / Light mode with a toggle (state lifted to App)
- Hero section with profile image and social links
- Projects grid with links and thumbnails
- Timeline-style Overview (background, experience, education)
- Skills cards with progress bars and external docs links (Simple Icons CDN)
- Contact section with mailto and social links
- Resume download link
- Built with accessibility and performance in mind

## Tech stack

- React (Vite)
- Tailwind CSS
- lucide-react (icons)
- External SVGs for skill logos (simple-icons CDN)
- Deployed static site friendly (Netlify / Vercel / GitHub Pages)

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shubhambatwal03/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

## Folder Structure

- `src/components/` – React components for the portfolio UI
- `src/App.jsx` – Main application logic
- `src/App.css` – Custom styles
